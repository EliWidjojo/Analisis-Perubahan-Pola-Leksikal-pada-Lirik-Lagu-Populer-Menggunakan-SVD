## Deskripsi
Proyek ini menganalisis lirik lagu populer Amerika Serikat dari tahun 1960 hingga 2025 menggunakan metode **Latent Semantic Analysis (LSA)** berbasis **Singular Value Decomposition (SVD)**. Analisis ini bertujuan untuk:

- Menemukan lagu paling mirip lintas tahun.
- Membandingkan tingkat similaritas dalam satu dekade dan antar dekade.
- Mengukur pergeseran leksikal antar dekade.

Data yang digunakan adalah **20 lagu teratas** dari Billboard Hot 100 Year-End Chart setiap tahunnya.

## Cara Menjalankan Program
1. Pastikan **Python 3.x** sudah terinstal.
2. Masuk ke folder `src` di terminal.
3. Instal dependensi jika ada menggunakan command "pip install -r requirements.txt"
4. Jalankan aplikasi dengan menggunakan command "python main.py"
